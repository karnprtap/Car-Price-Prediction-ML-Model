# Car-Price-Prediction-ML-Model


A Machine Learning web application that predicts the selling price of a used car based on its specifications. The application is built using **Python**, **Scikit-learn**, and **Streamlit**, providing a simple and interactive interface for users to estimate a car's market value.

---

## 📌 Project Overview

The value of a used car depends on several factors such as its brand, manufacturing year, fuel type, transmission, ownership history, and kilometers driven. This project applies a Machine Learning regression model to analyze these features and predict the estimated selling price.

---

## ✨ Features

- Predicts the selling price of used cars.
- Interactive web interface built with Streamlit.
- Machine Learning model trained on historical car data.
- Easy-to-use and responsive interface.
- Instant price prediction.

## 🚀 Installation

### Clone the Repository

```bash
git clone https://github.com/your-username/Car-Price-Prediction.git
```

### Navigate to the Project Folder

```bash
cd Car-Price-Prediction
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Run the Application

```bash
streamlit run app.py
```
